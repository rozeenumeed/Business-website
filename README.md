# Business Website

🚀 A modern and responsive Business Website built using HTML, CSS, and Bootstrap.

## Features
- Responsive Navigation Bar
- Image Carousel / Slider
- About Us Section
- Services Section
- Team Members Section
- Contact Section
- Mobile-Friendly Design

## Technologies Used
- HTML5
- CSS3
- Bootstrap
- Font Awesome

## Purpose
This project was created to practice front-end web development concepts, responsive layouts, and Bootstrap components while building a professional business website interface.
